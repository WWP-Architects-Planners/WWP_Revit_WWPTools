# 1. Project Setup

This page documents the tools found under this WWPTools panel.

## Line Type Tool
Location: WWPTools > 1. Project Setup > Line Type Tool
Screenshot: (Add later)
Purpose: Add all standard line types/patterns to your proejct. The file is located at "N:\Library\Design Software\Autodesk\Revit\Templates\Line Types\Linetypes.xlsm"
How to use:
1. Run the tool from the WWPTools tab and follow any prompts.
2. Review the results and undo if needed.

## Add Project Parameter (Pulldown)
Purpose: Import project parameters or create templates

### Create Template
Location: WWPTools > 1. Project Setup > Add Project Parameter > Create Template
Screenshot: (Add later)
Purpose: Create an Excel template with shared parameters listed in Variables sheet
How to use:
1. Have the target sheet(s) available or selected as prompted.
2. Prepare the Excel file you want to import or export.
3. Run the tool from the WWPTools tab and follow any prompts.
4. Review the results and undo if needed.

### Import from Excel
Location: WWPTools > 1. Project Setup > Add Project Parameter > Import from Excel
Screenshot: (Add later)
Purpose: Import project parameters from an Excel workbook based on shared parameters
How to use:
1. Prepare the Excel file you want to import or export.
2. Run the tool from the WWPTools tab and follow any prompts.
3. Review the results and undo if needed.

## Levels Setup (Pulldown)
Purpose: Manage Levels

### Levels Setup
Location: WWPTools > 1. Project Setup > Levels Setup > Levels Setup
Screenshot: (Add later)

How to use:
1. Run the tool from the WWPTools tab and follow any prompts.
2. Review the results and undo if needed.

### Views Creator
Location: WWPTools > 1. Project Setup > Levels Setup > Views Creator
Screenshot: (Add later)

3. Review the results and undo if needed.
