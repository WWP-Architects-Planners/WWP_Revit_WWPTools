# 5 Revit Cleanup

This page documents the tools found under this WWPTools panel.

## Project Cleaner

Location: WWPTools > 5. Revit Cleanup > Project Cleaner\
Screenshot:\
\
Purpose: Delete temp files, local files, and ACC caches.

How to use:

{% stepper %}
{% step %}
### Close and save all active files

Make sure no active documents are left unsaved.
{% endstep %}

{% step %}
### Open an empty dummy file

Open a blank Revit file to run the cleaner against.
{% endstep %}

{% step %}
### Run Project Cleaner and follow prompts

Execute the Project Cleaner tool and proceed through the on-screen prompts.
{% endstep %}
{% endstepper %}
