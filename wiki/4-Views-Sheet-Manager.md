# 4. Views Sheet Manager

This page documents the tools found under this WWPTools panel.

## Duplicate Views
Location: WWPTools > 4. Views Sheet Manager > Duplicate Views
Screenshot: (Add later)
Purpose: dupilicate selected views with new name, you can choose a text to replace with new name. if you leave it empty, it will be just '- copy'
How to use:
1. Select the target elements before running the tool.
2. Run the tool from the WWPTools tab and follow any prompts.
3. Review the results and undo if needed.

## Schedules to Excel/CSV
Location: WWPTools > 4. Views Sheet Manager > Schedules to Excel/CSV
Screenshot: (Add later)
Purpose: Publish existing schedules to excel file sheets or csv files.
How to use:
1. Have the target sheet(s) available or selected as prompted.
2. Prepare the Excel file you want to import or export.
3. Choose a folder for CSV output when prompted.
4. Run the tool from the WWPTools tab and follow any prompts.
5. Review the results and undo if needed.
